# 🎬 Netflix Content Analysis Project

## 📊 Overview

This project presents an end-to-end data analysis of Netflix content using **Python, SQL, and Tableau**. The goal is to uncover key insights related to content distribution, growth trends, ratings, and audience targeting.

From raw data cleaning to interactive dashboard creation, this project demonstrates a complete data analytics workflow.

---

## 🚀 Key Highlights

* Cleaned and transformed raw data using Python (Pandas)
* Performed structured data analysis using SQL
* Built an interactive dashboard in Tableau for data visualization
* Extracted meaningful insights on content trends and patterns

---

## 🎯 Objectives

* Analyze the distribution of Movies vs TV Shows
* Identify content growth trends over time
* Explore rating patterns and audience targeting
* Understand content characteristics such as duration and genre

---

## 🛠 Tools & Technologies

* Python (Pandas, NumPy)
* SQL (MySQL/PostgreSQL)
* Tableau (Dashboard & Storytelling)

---

## 📈 Outcome

The project provides actionable insights into Netflix’s content strategy, highlighting growth patterns, dominant genres, and audience preferences through data-driven analysis.
